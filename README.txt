aggregating_sakila
==================

This application provides some aggregation result of
`sakila database <http://dev.mysql.com/doc/sakila/en/index.html>`_.

Getting Started
---------------

- `import a sakila database <http://dev.mysql.com/doc/sakila/en/sakila-installation.html>`_

- cd <directory containing this file>

- $venv/bin/python setup.py develop

- $venv/bin/initialize_sakila_db development.ini

- $venv/bin/pserve development.ini -n mysql user=sameuser password=samepassword
